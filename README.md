### Project Overview

This project demonstrates a high-end, scroll-driven hero section built with modern web technologies. It focuses on fluid motion, premium aesthetics, and performance.

**Key Features:**
- **Scroll-Driven Motion:** A high-quality car object responds directly to scroll progress using GSAP ScrollTrigger.
- **Initial Load Stagger:** Staggered reveal of the headline "WELCOME ITZFIZZ" and impact metrics on page load.
- **Glassmorphic UI:** Modern impact cards with blurred backgrounds and vibrants accents.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile screens.

---

### Tech Stack Used

- **Framework:** Next.js 16 (Turbopack)
- **Library:** React 19
- **Animation:** GSAP (GreenSock Animation Platform) + GSAP ScrollTrigger
- **Styling:** Tailwind CSS 4
- **Typography:** Google Fonts (Outfit & Space Grotesk)

---

### How to Run Locally

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

---

### Performance Optimization

- Used GPU-accelerated CSS properties (`transform`, `opacity`) for all scroll-based animations.
- Prevented potential layout reflows by avoiding height/width mutations during scroll.
- Optimized for static export with `unoptimized` images for GitHub Pages compatibility.
